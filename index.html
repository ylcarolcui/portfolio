Zoo Doc

ZooDoc is a website that allows users to find and rate veterinarians in their metro areas. Users may create accounts, log in, read, rate, update, and delete vet reviews.

It's a full stack application with CRUD functionality for veterinarian reviews. The frontend was built with ReactJS Framework and the backend was built with Django REST Framework with PostgreSQL database.

Technologies used: HTML, CSS, JavaScript, React, Django, Bootstrap, Python, SQL
